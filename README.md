# sholat-api

Local Node.js API to serve `gerakan` and `bacaan` from your MySQL database.

Setup

1. Install dependencies:

```bash
npm install
```

2. Copy `.env.example` to `.env` and fill your database credentials.

3. Run the server:

```bash
npm start
```

Endpoints

- `GET /api/gerakan` — list gerakan
- `GET /api/bacaan?id_gerakan=ID` — list bacaan for a gerakan
